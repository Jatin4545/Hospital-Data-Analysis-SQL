
🏥 Hospital Management Data Analysis – Complete SQL Case Study

## 📌 Project Overview

This project focuses on analyzing a **Hospital Management Dataset** using SQL to extract meaningful business insights related to patient behavior, doctor workload, treatment costs, and hospital revenue.

The main objective of this project is to perform **Exploratory Data Analysis (EDA)** using SQL queries and help hospital management in making data-driven decisions such as:

* Resource Allocation
* Doctor Workload Management
* Patient Demographic Analysis
* Treatment Cost Monitoring
* Revenue Contribution Analysis
* Appointment Trend Identification

This project demonstrates real-world use of SQL in the Healthcare Analytics domain.

---

## 🎯 Business Objectives

The key business problems addressed in this analysis include:

* Identifying most demanded doctor specialization
* Analyzing patient age distribution
* Monitoring doctor workload based on appointment volume
* Finding high-cost treatments that require review
* Understanding patient registration trends over time
* Identifying top revenue contributing patients
* Analyzing appointment status distribution
* Finding the most frequently performed treatments
* Segmentation of doctors based on experience level
* Total revenue generated by hospital

---

## 🗂️ Dataset Information

The dataset consists of the following relational tables:

| Table Name   | Description                        |
| ------------ | ---------------------------------- |
| Patients     | Patient demographic details        |
| Doctors      | Doctor specialization & experience |
| Appointments | Appointment records                |
| Treatments   | Treatment type and cost            |
| Billing      | Payment and revenue details        |

## 📊 Analysis Performed

The following SQL concepts are used in this project:

* Joins (Inner Join, Left Join)
* Common Table Expressions (CTEs)
* CASE Statements
* Window Functions (RANK, DENSE_RANK)
* Subqueries
* Group By & Having Clause
* Date Functions
* String Functions
